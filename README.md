# You Are An Agent

A game to build empathy for AI agents by experiencing their constraints firsthand.

**Play it:** [youareanagent.app](https://youareanagent.app)

**Read more:** [AX: Building Empathy and Intuition for Agents](https://robkopel.com/field-notes/ax-agent-experience.html)

## Run Locally

```bash
npm install
npm run dev
```

Set `GEMINI_API_KEY` in `.env.local` for AI features.
